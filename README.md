# 书签修复与比对工具 (Bookmark Fixer)

这是一个轻量级、完全在本地浏览器中运行的 Web 工具，旨在帮助用户高效地管理、修复和比对浏览器书签。

## 主要功能
- **智能对比**：通过上传旧书签文件，自动识别并定位缺失的网站。
- **同名匹配逻辑**：即使 URL 发生微小变化，只要名称匹配，工具即视为已存在，有效减少误报。
- **本地化隐私安全**：所有数据处理均在你的浏览器内存中完成，**没有任何数据会上传到服务器**，确保你的书签隐私安全。
- **即开即用**：无需安装任何复杂软件，直接在现代浏览器中打开即可使用。

## 如何使用
1. **在线访问**：直接通过 [GitHub Pages 链接](https://你的用户名.github.io/bookmark-fixer/) 打开。
2. **导入文件**：点击“上传旧书签”按钮，导入你的备份文件。
3. **粘贴内容**：在文本框中粘贴你需要修复的新书签列表。
4. **一键对比**：系统将自动对比并显示缺失列表。

## 技术栈
- HTML5 / JavaScript (原生实现)
- Tailwind CSS (用于界面 UI)

## 关于隐私
本工具纯前端开发，所有操作均在本地进行。开发者无法获取、访问或存储你的任何书签数据。
